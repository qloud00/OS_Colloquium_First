# OS_Colloquium_1

Проект содержит реализацию трех алгоритмических задач на языке C++ с применением современных практик разработки, модульного тестирования и систем сборки.

## Технологический стек

*   **Язык:** C++20
*   **Система сборки:** CMake
*   **Тестирование:** Google Test (gtest)

## Описание задач

Проект разделен на независимые модули, соответствующие поставленным задачам:

### 1. Генерация последовательности факториалов
**Модуль:** `factorial`

Разработать функцию, которая принимает на вход натуральное число `n` и возвращает контейнер, содержащий первые `n` факториалов.
*   Реализована валидация входных данных (обработка отрицательных чисел).
*   Добавлена проверка на переполнение типа `uint64_t`.

### 2. Удаление дубликатов с сохранением порядка
**Модуль:** `dupl_erase`

Разработать функцию, которая принимает на вход контейнер (массив) с числами и возвращает новый контейнер с неизменным порядком первого появления элементов, но без дубликатов.
*   Алгоритмическая сложность: O(N) по времени и памяти.
*   Использована хеш-таблица (`std::unordered_set`) для оптимизации поиска.

### 3. Рекурсивный реверс связного списка
**Модуль:** `list_reverse`

Разработать структуру односвязного списка и реализовать функцию разворота (инверсии) списка, используя рекурсию.
*   Реализован шаблонный класс списка.
*   Обеспечена безопасность работы с памятью (запрет неявного копирования).

## Структура проекта

*   `factorial/` — исходный код и тесты для задачи с факториалами.
*   `dupl_erase/` — исходный код и тесты для задачи удаления дубликатов.
*   `list_reverse/` — реализация связного списка и тесты.
*   `utils/` — общая библиотека вспомогательных утилит (безопасный ввод данных).
